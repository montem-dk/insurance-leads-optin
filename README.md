# Insurance Leads Opt-In Page

Opt-in → Thank You → Google Doc funnel for life insurance lead generation.

## Setup

1. Replace `GOOGLE_DOC_URL` and `WEBHOOK_URL` in `src/App.jsx`
2. `npm install`
3. `npm run dev` (local preview)
4. `npm run build` (production build in `dist/`)

## Deploy to Vercel

```bash
npx vercel
```
